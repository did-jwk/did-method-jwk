did:jwk Method Specification

This DID method specifies a non-registry approach for representing JSON Web Keys [[JWK]] as DID identifiers.

Introduction

JSON Web Keys [[JWK]] is a popular data structure for representing cryptographic keys using JavaScript Object Notation (JSON) [[JSON]]. The did:jwk method can be used for represting JWK-encoded public keys as Decentralized Identifiers (DIDs) [[DID-CORE]]

The did:jwk Format

The format for the did:jwk method consists of the did:jwk prefix, followed by the base64url [[rfc4648]] encoded representation of the JWK public key according to [[JWK]].

Examples

Ed25519

An example of an Ed25519 public key encoded in JWK is: 

            {
                "kty": "OKP",
                "crv": "Ed25519",
                "x": "lwIUhDZMC4bnohWh1rFAsMrS1V6k-fu0ck1uoUge8EQ"
            }

The representation of this key using did:jwk is: 

                did:jwk:ewogICJrdHkiOiAiT0tQIiwKICAiY3J2IjogIkVkMjU1MTkiLAogICJ4IjogImx3SVVoRFpNQzRibm9oV2gxckZBc01yUzFWNmstZnUwY2sxdW9VZ2U4RVEiCn0
